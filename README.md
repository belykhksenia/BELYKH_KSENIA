# Система управления базой данных студентов

## Описание проекта

Проект представляет собой объектно-ориентированную систему управления данными студентов с консольным интерфейсом. 
Реализованы все CRUD операции (Create, Read, Update, Delete) с валидацией данных и обработкой исключений.

## Основные возможности

1. Добавление записи - создание новой записи о студенте
2. Просмотр всех записей - отображение всех студентов в базе
3. Поиск по фильтру - гибкий поиск по любому полю
4. Обновление записи - изменение данных существующего студента
5. Удаление записи - удаление записи с подтверждением
6. Сортировка записей - можно сортировать по нескольким параментрам

### Валидация данных

- Проверка уникальности ID студента
- Проверка корректности возраста (неотрицательное значение)
- Обработка пустых полей при обновлении
- Обработка некорректного ввода

### Установка и запуск

1. Клонируйте репозиторий:
git clone <repository-url>
cd <project-directory>

2. Запустите приложение
python -m src.db

## Использование программы

### После запуска появится главное меню:

=== База студентов ===
1. Добавить запись
2. Показать все записи
3. Найти записи по фильтру
4. Обновить запись
5. Удалить запись
6. Сортировать записи
0. Выход


### Навигация:

· Введите номер пункта меню (0-6) и нажмите Enter
· Для отмены фильтров при поиске оставляйте поля пустыми
· При обновлении пустые поля означают "оставить без изменений"
