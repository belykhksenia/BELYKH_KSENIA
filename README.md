# Описание функциональности

Проект представляет собой консольное приложение для управления базой данных студентов, работающее полностью в оперативной памяти.

Реализованные возможности:

1. Создание записи (Create)
   · Добавление нового студента с полями: ID, имя, фамилия, возраст, пол
   · Проверка уникальности ID
   · Валидация возраста (не может быть отрицательным)
2. Чтение записей (Read)
   · Просмотр всех студентов
   · Поиск студентов по фильтрам:
     · ID
     · Имя
     · Фамилия
     · Возраст
     · Пол
3. Обновление записей (Update)
   · Изменение данных существующего студента по ID
   · Возможность обновить только выбранные поля
4. Удаление записей (Delete)
   · Удаление студента по ID
   · Подтверждение операции перед удалением

Типы данных:

· Запись студента представлена кортежем (id, first_name, second_name, age, sex)
· Данные хранятся в глобальном списке Student


Инструкция по запуску

Требования

· Python 3.6 или выше
· Любая операционная система (Windows/Linux/macOS)
